# Pokopia Planner

Plan your dream cloud island in Pokopia - a Pokemon x Animal Crossing-style game.

## Features

- **Island Planner** - Drag-and-drop 16x16 grid to place buildings, decorations, terrain, and Pokemon habitats
- **Progress Tracker** - Checklist with categories for buildings, resources, island goals, and Pokemon
- **Shareable Links** - Share your island layout with friends via compressed URL

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start planning.

## Tech Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS v4
- @dnd-kit for drag-and-drop
- lz-string for URL compression
